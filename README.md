# Алгоритмы

## Постановка

Реализация следующих алгоритмов:
1. Сортировки.
    1. Обменные:
		- пузырьковая;
		- шейкерная;
		- гномья;
		- быстрая-Хоара-разделением.
    2. Включением-вставками (вставками, Шелла).
	3. Выбором (простым выбором, пирамидальная).
	4. Слиянием (слиянием).
2. Алгоритмы на графах (поиска кротчайшего пути, ...).
3. Поиск в массивах.
4. Структуры данных (скорее про деревья. все виды сетевых структур данных).
